# Guia3-Swift

Atividade prática 3 para a matéria de Desenvolvimento Mobile IOS.

Enums com valores associados modelam estados complexos, enquanto closures permitem lógica flexível e reutilizável (inclusive com controle de memória).
Generics deixam o código reutilizável e seguro pra qualquer tipo, evitando duplicação.
Protocolos com extensions reforçam o padrão moderno do Swift: código desacoplado, reutilizável e orientado a comportamento.
